# Input and Output Functions in Python

## Overview

Input and Output (I/O) operations are essential in any programming language. They allow a program to interact with users by taking input and displaying output.

In Python:
- Input is taken using the `input()` function
- Output is displayed using the `print()` function

---

## Output in Python

### print() Function

The `print()` function is used to display output on the screen.

### Syntax:
```python
print(value1, value2, ..., sep=' ', end='\n')
````

### Parameters:

* `value1, value2`: Values to be printed
* `sep`: Separator between values (default is space)
* `end`: What to print at the end (default is newline)

---

### Example 1: Basic Output

```python
print("Hello, World!")
```

---

### Example 2: Multiple Values

```python
name = "Chirag"
age = 25

print("Name:", name, "Age:", age)
```

---

### Example 3: Using sep Parameter

```python
print("2026", "03", "24", sep="-")
```

Output:

```
2026-03-24
```

---

### Example 4: Using end Parameter

```python
print("Hello", end=" ")
print("World")
```

Output:

```
Hello World
```

---

### Example 5: Formatted Output (f-strings)

```python
name = "Chirag"
age = 25

print(f"My name is {name} and I am {age} years old")
```

---

## Input in Python

### input() Function

The `input()` function is used to take input from the user.

### Syntax:

```python
input("message")
```

* The message is optional (prompt shown to the user)
* Input is always returned as a string

---

### Example 1: Basic Input

```python
name = input("Enter your name: ")
print("Hello", name)
```

---

### Example 2: Taking Numeric Input

Since `input()` returns a string, type conversion is required.

```python
age = int(input("Enter your age: "))
print("Your age is", age)
```

---

### Example 3: Taking Float Input

```python
price = float(input("Enter price: "))
print("Price is", price)
```

---

### Example 4: Multiple Inputs

```python
a = int(input("Enter first number: "))
b = int(input("Enter second number: "))

sum = a + b
print("Sum =", sum)
```

---

### Example 5: Taking Multiple Values in One Line

```python
a, b = map(int, input("Enter two numbers: ").split())
print("Sum =", a + b)
```

---

## Type Conversion in Input

Since input is always a string, conversion is necessary.

### Common Conversions:

```python
int(input())
float(input())
str(input())
```

---

## Combining Input and Output

```python
name = input("Enter your name: ")
age = int(input("Enter your age: "))

print(f"Hello {name}, you are {age} years old")
```

---

## Common Errors

### 1. Invalid Conversion

```python
age = int(input("Enter age: "))
```

If user enters text instead of number:

```
ValueError: invalid literal for int()
```

---

### 2. Missing Input

```python
num = int(input())
```

If nothing is entered, it may cause unexpected behavior.

---

## Best Practices

* Always validate user input
* Use meaningful prompts
* Convert input to correct data type
* Use formatted strings (f-strings) for better readability

---

## Practice Examples

### Example 1: Simple Calculator

```python
a = int(input("Enter first number: "))
b = int(input("Enter second number: "))

print("Addition =", a + b)
```

---

### Example 2: Area of Circle

```python
radius = float(input("Enter radius: "))
area = 3.14 * radius * radius

print("Area =", area)
```

---

## Conclusion

Input and Output functions are fundamental to Python programming. The `input()` function allows user interaction, while `print()` helps display results. Understanding how to use these functions effectively is essential for building interactive applications.
