# Type Casting in Python

## Overview

Type casting (also known as type conversion) is the process of converting one data type into another. It is commonly used when performing operations that require a specific data type.

In Python, type casting can be:
- Implicit (automatic)
- Explicit (manual)

---

## Why Type Casting is Important

- To perform mathematical operations
- To handle user input correctly
- To ensure compatibility between different data types
- To avoid runtime errors

---

## Types of Type Casting

### 1. Implicit Type Casting

Python automatically converts one data type into another without user intervention.

### Example:

```python
x = 10      # int
y = 2.5     # float

result = x + y
print(result)
print(type(result))
````

### Explanation:

* Python converts `int` to `float`
* Output will be a float value

---

### 2. Explicit Type Casting

The programmer manually converts one data type into another using built-in functions.

---

## Common Type Casting Functions

| Function | Description                  |
| -------- | ---------------------------- |
| int()    | Converts value to integer    |
| float()  | Converts value to float      |
| str()    | Converts value to string     |
| bool()   | Converts value to boolean    |
| list()   | Converts value to list       |
| tuple()  | Converts value to tuple      |
| set()    | Converts value to set        |
| dict()   | Converts value to dictionary |

---

## Integer Casting

### Example:

```python
x = "10"
y = int(x)

print(y)
print(type(y))
```

---

## Float Casting

### Example:

```python
x = "3.14"
y = float(x)

print(y)
print(type(y))
```

---

## String Casting

### Example:

```python
x = 100
y = str(x)

print(y)
print(type(y))
```

---

## Boolean Casting

### Example:

```python
print(bool(1))      # True
print(bool(0))      # False
print(bool(""))     # False
print(bool("Hi"))   # True
```

---

## List Casting

### Example:

```python
x = (1, 2, 3)
y = list(x)

print(y)
print(type(y))
```

---

## Tuple Casting

### Example:

```python
x = [1, 2, 3]
y = tuple(x)

print(y)
print(type(y))
```

---

## Set Casting

### Example:

```python
x = [1, 2, 2, 3]
y = set(x)

print(y)
print(type(y))
```

---

## Dictionary Casting

### Example:

```python
x = [("a", 1), ("b", 2)]
y = dict(x)

print(y)
print(type(y))
```

---

## Type Casting with Input

Since `input()` returns a string, casting is required.

### Example:

```python
age = int(input("Enter your age: "))
print("Age:", age)
```

---

## Multiple Type Casting

### Example:

```python
a = "10"
b = "20"

sum = int(a) + int(b)
print("Sum =", sum)
```

---

## Type Casting in Expressions

### Example:

```python
x = 5
y = "10"

result = x + int(y)
print(result)
```

---

## Invalid Type Casting

### Example:

```python
x = "Hello"
y = int(x)   # Error
```

### Error:

```
ValueError: invalid literal for int()
```

---

## Best Practices

* Always validate data before casting
* Use try-except to handle errors
* Avoid unnecessary conversions
* Use appropriate data types for operations

---

## Example with Exception Handling

```python
try:
    num = int(input("Enter a number: "))
    print("Number:", num)
except ValueError:
    print("Invalid input! Please enter a number.")
```

---

## Summary

* Type casting converts one data type into another
* Two types: implicit and explicit
* Python provides built-in functions for conversion
* Proper casting avoids errors and improves program reliability
