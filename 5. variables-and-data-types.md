
# Variables and Data Types in Python

## Overview

Variables are used to store data in a program, and data types define the type of data a variable can hold. Python is dynamically typed, meaning you do not need to declare the type of a variable explicitly.

---

## What is a Variable?

A variable is a name given to a memory location where data is stored.

### Example:
```python
name = "Chirag"
age = 25
````

### Rules for Naming Variables

* Must start with a letter or underscore (`_`)
* Cannot start with a number
* Can contain letters, numbers, and underscores
* Case-sensitive (`name` and `Name` are different)
* Cannot use reserved keywords (like `if`, `for`, `while`)

---

## What are Data Types?

Data types specify the kind of value a variable holds. Python has several built-in data types.

---

## 1. Numeric Data Types

### Integer (int)

Used to store whole numbers.

```python
x = 10
y = -5
print(type(x))
```

---

### Float (float)

Used to store decimal numbers.

```python
pi = 3.14
price = 99.99
print(type(pi))
```

---

### Complex (complex)

Used for complex numbers.

```python
z = 2 + 3j
print(type(z))
```

---

## 2. String (str)

Used to store text data.

```python
name = "Python"
message = 'Hello World'
print(type(name))
```

---

## 3. Boolean (bool)

Represents True or False values.

```python
is_active = True
is_logged_in = False
print(type(is_active))
```

---

## 4. List (list)

An ordered, mutable collection of elements.

```python
numbers = [1, 2, 3, 4]
names = ["Alice", "Bob", "Charlie"]
print(type(numbers))
```

---

## 5. Tuple (tuple)

An ordered, immutable collection of elements.

```python
coordinates = (10, 20)
colors = ("red", "green", "blue")
print(type(coordinates))
```

---

## 6. Set (set)

An unordered collection of unique elements.

```python
unique_numbers = {1, 2, 3, 3, 4}
print(unique_numbers)
print(type(unique_numbers))
```

---

## 7. Dictionary (dict)

A collection of key-value pairs.

```python
student = {
    "name": "Chirag",
    "age": 21,
    "course": "Python"
}
print(type(student))
```

---

## Dynamic Typing in Python

Python allows changing the data type of a variable at runtime.

```python
x = 10
print(type(x))

x = "Hello"
print(type(x))
```

---

## Type Checking

Use `type()` to check the data type of a variable.

```python
x = 100
print(type(x))
```

---

## Type Casting (Type Conversion)

Convert one data type into another.

```python
x = "10"
y = int(x)

print(y)
print(type(y))
```

---

## Multiple Variable Assignment

```python
a, b, c = 1, 2, 3
print(a, b, c)
```

---

## Constants in Python

Python does not have true constants, but we use uppercase variable names by convention.

```python
PI = 3.14
GRAVITY = 9.8
```

## Conclusion

Variables and data types are the foundation of Python programming. Understanding how to store and manipulate different types of data is essential for writing efficient programs. Python’s dynamic typing makes it flexible and easy to use for beginners.
