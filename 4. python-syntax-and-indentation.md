
# Python Syntax & Indentation

## Overview

Python syntax refers to the set of rules that define how Python programs are written and interpreted. One of the most important aspects of Python syntax is indentation, which defines the structure and flow of the program.

Unlike many other programming languages that use braces `{}` to define code blocks, Python uses indentation (spaces or tabs).

---

## What is Python Syntax?

Python syntax is designed to be simple, readable, and close to the English language. It allows developers to write clean and understandable code.

### Key Points of Python Syntax
- No need for semicolons (`;`) at the end of statements
- Statements are written line by line
- Case-sensitive language (`name` and `Name` are different)
- Indentation is mandatory for defining blocks

---

## What is Indentation?

Indentation refers to the spaces at the beginning of a line of code.

In Python:
- Indentation is used to define blocks of code
- It replaces curly braces `{}` used in other languages
- Standard practice is to use **4 spaces**

---

## Why Indentation is Important?

- Defines scope of loops, functions, and conditions
- Improves readability
- Mandatory in Python (not optional)

Incorrect indentation will result in an error:
```

IndentationError: expected an indented block

````

---

## Rules of Indentation

- Use consistent spacing (prefer 4 spaces)
- Do not mix tabs and spaces
- All statements in the same block must have equal indentation

---

## Example 1: Simple If Statement

```python
age = 18

if age >= 18:
    print("You are eligible to vote")
````

### Explanation

* The `print()` statement is indented inside the `if` block
* If indentation is removed, Python will throw an error

---

## Example 2: If-Else Statement

```python
number = 10

if number % 2 == 0:
    print("Even number")
else:
    print("Odd number")
```

### Explanation

* Both `if` and `else` blocks have proper indentation
* Each block contains its own set of statements

---

## Example 3: Nested If Statement

```python
marks = 75

if marks >= 50:
    print("Pass")
    if marks >= 70:
        print("Distinction")
```

### Explanation

* The second `if` is nested inside the first
* Indentation increases for inner blocks

---

## Example 4: For Loop

```python
for i in range(5):
    print("Number:", i)
```

### Explanation

* The loop body is indented
* All statements inside the loop must align

---

## Example 5: Function with Indentation

```python
def greet(name):
    print("Hello", name)
    print("Welcome to Python")

greet("Chirag")
```

### Explanation

* Function body is indented
* All lines inside the function must have the same indentation

---

## Incorrect Indentation Example

```python
if True:
print("Hello")
```

### Error

```
IndentationError: expected an indented block
```

---

## Best Practices

* Always use 4 spaces for indentation
* Use code editors like VS Code or PyCharm (they handle indentation automatically)
* Keep code clean and properly aligned
* Avoid mixing tabs and spaces

---

## Conclusion

Python syntax is simple and easy to learn, but indentation plays a critical role in defining program structure. Proper indentation ensures that the code runs correctly and remains readable. Mastering indentation is essential for writing efficient Python programs.
