
# Case Study: Print Personal Details using Python

## Overview

This case study demonstrates how to create a simple Python program that collects and displays personal details of a user. It helps beginners understand input, output, variables, and formatting.

---

## Problem Statement

Design a Python program that:
- Accepts personal details from the user
- Stores the data using variables
- Displays the information in a structured format

---

## Real-World Scenario

Consider a situation where a company or educational institute wants to collect basic details of a user during registration. The system should:
- Take user input
- Store the information
- Display it in a readable format

---

## Data to be Collected

- Full Name  
- Age  
- Gender  
- Email  
- Phone Number  
- City  

---

## Approach

1. Use `input()` to collect data from the user  
2. Convert data types where required (e.g., age to integer)  
3. Store values in variables  
4. Use `print()` to display formatted output  

---

## Complete Python Code

```python
# Personal Details Program

# Taking input from user
name = input("Enter your full name: ")
age = int(input("Enter your age: "))
gender = input("Enter your gender: ")
email = input("Enter your email: ")
phone = input("Enter your phone number: ")
city = input("Enter your city: ")

# Displaying details
print("\n----- Personal Details -----")
print("Name       :", name)
print("Age        :", age)
print("Gender     :", gender)
print("Email      :", email)
print("Phone No   :", phone)
print("City       :", city)
````

---

## Sample Input

```
Enter your full name: Chirag Raul
Enter your age: 25
Enter your gender: Male
Enter your email: chirag@example.com
Enter your phone number: 9876543210
Enter your city: Pune
```

## Improved Version (Using f-Strings)

```python
# Personal Details using f-strings

name = input("Enter your full name: ")
age = int(input("Enter your age: "))
gender = input("Enter your gender: ")
email = input("Enter your email: ")
phone = input("Enter your phone number: ")
city = input("Enter your city: ")

print(f"""
----- Personal Details -----
Name       : {name}
Age        : {age}
Gender     : {gender}
Email      : {email}
Phone No   : {phone}
City       : {city}
""")
```

---

## Explanation

* `input()` is used to collect user data
* `int()` converts age into integer
* Variables store user information
* `print()` displays formatted output
* f-strings make output cleaner and more readable

---

## Validation Version (Advanced Beginner)

```python
try:
    name = input("Enter your full name: ")
    age = int(input("Enter your age: "))
    gender = input("Enter your gender: ")
    email = input("Enter your email: ")
    phone = input("Enter your phone number: ")
    city = input("Enter your city: ")

    print(f"\n{name}, your details have been recorded successfully.")
except ValueError:
    print("Invalid input! Age must be a number.")
```
