
# Case Study: Simple Calculator (All Operations)

## Overview

This case study demonstrates how to build a simple calculator using Python. The program performs basic arithmetic operations such as addition, subtraction, multiplication, and division.

This project is designed using only the topics covered so far:
- Input and Output
- Variables
- Data Types
- Type Casting
- Conditional Statements

---

## Problem Statement

Design a Python program that:
- Takes two numbers as input from the user
- Allows the user to choose an operation
- Performs the selected operation
- Displays the result

---

## Real-World Scenario

In daily life, calculators are used for quick mathematical calculations. This program simulates a basic calculator used in:
- Shops for billing
- Students for solving problems
- Office work for quick calculations

---

## Operations Covered

- Addition (+)
- Subtraction (-)
- Multiplication (*)
- Division (/)

---

## Approach

1. Take input from the user
2. Convert input into numeric type using type casting
3. Ask the user to choose an operation
4. Use conditional statements (`if-elif-else`) to perform the operation
5. Display the result

---

## Complete Python Code

```python
# Simple Calculator Program

# Taking input
num1 = float(input("Enter first number: "))
num2 = float(input("Enter second number: "))

print("\nSelect Operation:")
print("1. Addition (+)")
print("2. Subtraction (-)")
print("3. Multiplication (*)")
print("4. Division (/)")

choice = input("Enter choice (1/2/3/4): ")

# Performing operation
if choice == '1':
    result = num1 + num2
    print("Result:", result)

elif choice == '2':
    result = num1 - num2
    print("Result:", result)

elif choice == '3':
    result = num1 * num2
    print("Result:", result)

elif choice == '4':
    if num2 != 0:
        result = num1 / num2
        print("Result:", result)
    else:
        print("Error: Division by zero is not allowed")

else:
    print("Invalid choice")
````

---
