# Installing Python & IDEs (VS Code, PyCharm)

## Overview

This guide will help you install:

* Python (latest version)
* Visual Studio Code (VS Code)
* PyCharm IDE

It also includes verification steps and recommended extensions.

---

# 1. Install Python (Latest Version)

## Step 1: Download Python

* Go to the official website:
  [https://www.python.org/downloads/](https://www.python.org/downloads/)

* Click on **Download Python (Latest Version)**
  (Example: Python 3.x.x)

---

## Step 2: Run Installer

### Important (Must Do)

* Check the box: **Add Python to PATH**

### Then click:

* **Install Now**

---

## Step 3: Verify Installation

Open Command Prompt / Terminal and run:

```bash
python --version
```

or

```bash
python3 --version
```

You should see the installed Python version.

---

## Step 4: Verify pip (Package Manager)

```bash
pip --version
```

---

## Step 5: Upgrade pip (Recommended)

```bash
python -m pip install --upgrade pip
```

---

# 2. Install Visual Studio Code (VS Code)

## Step 1: Download VS Code

* Official website:
  [https://code.visualstudio.com/](https://code.visualstudio.com/)

* Click **Download for Windows / Mac / Linux**

---

## Step 2: Install VS Code

During installation:

* Check **Add to PATH**
* Check **Add "Open with Code" option**

Click **Install**

---

## Step 3: Install Python Extension

Open VS Code → Go to Extensions (Ctrl + Shift + X)

Search and install:

* Python (by Microsoft)
* Pylance (recommended for IntelliSense)

---

## Step 4: Configure Python Interpreter

* Press `Ctrl + Shift + P`
* Search: **Python: Select Interpreter**
* Choose your installed Python version

---

## Step 5: Run Python Program

Create a file:

```python
print("Hello, Python!")
```

Run using:

* Right-click → Run Python File
  OR
* Terminal:

```bash
python filename.py
```

---

# 3. Install PyCharm IDE

## Step 1: Download PyCharm

* Official website:
  [https://www.jetbrains.com/pycharm/download/](https://www.jetbrains.com/pycharm/download/)

Choose:

* Community Edition (Free)
* Professional Edition (Paid)

---

## Step 2: Install PyCharm

During installation:

* Check **Add to PATH**
* Associate `.py` files

Click **Install**

---

## Step 3: Launch PyCharm

* Open PyCharm
* Click **New Project**

---

## Step 4: Configure Python Interpreter

* Choose **New environment using Virtualenv** (recommended)
* Select Base Interpreter (installed Python)

---

## Step 5: Create First Program

Create a file:

```python
print("Hello from PyCharm!")
```

Click **Run**

---

# 4. Recommended Setup (Best Practice)

## Create Virtual Environment

```bash
python -m venv myenv
```

Activate:

### Windows

```bash
myenv\Scripts\activate
```

### Mac/Linux

```bash
source myenv/bin/activate
```

---

## Install Required Packages

```bash
pip install numpy pandas matplotlib
```

---

# 5. Common Issues & Fixes

## Python Not Recognized

* Reinstall Python
* Ensure **Add to PATH** is checked

## pip Not Working

```bash
python -m ensurepip --upgrade
```

## VS Code Not Detecting Python

* Use: `Python: Select Interpreter`
