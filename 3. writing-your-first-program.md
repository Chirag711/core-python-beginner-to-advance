# Writing Your First Python Program (Hello World)



## Overview

The "Hello World" program is the first step in learning any programming language. It helps you understand how to write, save, and execute a basic Python program.

---

## Prerequisites

Before starting, make sure:
- Python is installed on your system
- VS Code or PyCharm is installed (optional but recommended)

---

## Step 1: Open Your Editor

You can use any of the following:
- Notepad (basic)
- VS Code (recommended)
- PyCharm (IDE)

---

## Step 2: Create a Python File

- Create a new file
- Save it with `.py` extension

Example:
```

hello.py

````

---

## Step 3: Write Your First Program

Type the following code in your file:

```python
print("Hello, World!")
````

### Explanation:

* `print()` is a built-in function in Python
* It is used to display output on the screen
* `"Hello, World!"` is a string (text)

---

## Step 4: Save the File

* Press `Ctrl + S` (Windows/Linux)
* Press `Cmd + S` (Mac)

Make sure the file is saved with `.py` extension

---

## Step 5: Run the Program

### Method 1: Using Terminal / Command Prompt

1. Open Terminal or Command Prompt
2. Navigate to the folder where your file is saved

Example:

```bash
cd path_to_your_folder
```

3. Run the program:

```bash
python hello.py
```

OR (in some systems):

```bash
python3 hello.py
```

---

### Method 2: Using VS Code

* Right-click inside the file
* Click on **Run Python File**
  OR
* Click the ▶ Run button

---

### Method 3: Using PyCharm

* Right-click the file
* Click **Run 'hello'**

---

## Step 6: Output

You will see the following output:

```
Hello, World!
```

---

## Common Mistakes

### 1. Missing Quotes

```python
print(Hello, World!)
```

Error: Strings must be inside quotes

### 2. Incorrect File Extension

* File should be `hello.py`, not `hello.txt`

### 3. Python Not Recognized

* Ensure Python is added to PATH

---

